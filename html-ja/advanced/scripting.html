Pythonによるスクリプト作成¶

前のチュートリアルでは、シミュレーションプロジェクトをXML形式で記述しましたが、Pythonスクリプトを使ってシミュレーションを設定することもできます。

以下はスクリプトの例です:

import hrpsys
import commands

openhrp_dir = commands.getoutput("pkg-config --variable=prefix openhrp3.1")
model_dir="file://"+openhrp_dir+"/share/OpenHRP-3.1/sample/model/"

sim = hrpsys.Simulator()
sim.initViewer()
box  = sim.loadBody("box",  model_dir+"box.wrl")
box.rootLink().R = [0,0,0]
box2 = sim.loadBody("box2", model_dir+"box2.wrl")
sim.addCollisionCheckPair(box, box2)
sim.initialize()
sim.simulate(1.0)

このスクリプトは:

  • hrpsysライブラリをインポート（１行目）

  • pkg-configコマンドを子プロセスとして起動しOpenHRPサンプルフォルダを取得（4-5行目）

  • シミュレータのロード（7-8行目）

  • ２つの箱を作成し、位置を設定する（9-11行目）

  • 箱の間の干渉チェックを設定する（12行目）

  • シミュレーションを１秒間実行する（13-14行目）

import hrpsys

sim = hrpsys.Simulator()
sim.initViewer()

body = sim.createBody("body")

root = body.rootLink()
s1 = root.addCube(0.2, 1.0, 0.2)
s1.diffuse = [1,0,0,1]

child = root.addChildLink("child")
child.b = [0,0.5,0]
child.jointType = "rotate"
child.a = [1,0,0]
child.m = 1.0
child.c = [0,0.5,0]
child.I = [1,0,0,0,1,0,0,0,1]
s2 = child.addCube(0.5, 1.0, 0.1)
s2.b = [0,0.5,0]
s2.diffuse = [0,1,0,1]

sim.initialize()
sim.simulate(3)

このスクリプトは:

  • hrpsysライブラリをインポート（１行目）

  • シミュレータをロードする（3-4行目）

  • プリミティブを使って赤色で箱型のモデルを作成する（6-10行目）

  • 関節を作成し各物理パラメータを設定する（12-18行目）

  • ２つ目の箱を作成し関節で接続する（19-21行目）

  • シミュレーションを初期化し３秒間実行する（23-24行目）
